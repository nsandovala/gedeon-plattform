# Init Discord Bot Playbook

## Objetivo
Crear un bot mínimo y coherente con la fase actual.

## Reglas
- no meter lógica crítica de negocio
- no crear comandos innecesarios
- no depender de estado oculto
- preparar futura conexión con API

## Pasos
1. leer `CONTEXT.md`
2. inicializar app del bot
3. crear estructura mínima
4. registrar comandos MVP
5. documentar variables necesarias
6. dejar listo para integración futura
