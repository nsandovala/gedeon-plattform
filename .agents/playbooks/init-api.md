# Init API Playbook

## Objetivo
Crear la base limpia de la API sin sobrediseño.

## Reglas
- FastAPI
- PostgreSQL
- sin Firebase como base principal
- sin OCR
- sin microservicios

## Pasos
1. leer `CONTEXT.md`
2. crear estructura base
3. definir configuración
4. preparar conexión a DB
5. crear módulo health
6. definir primer módulo real según fase
