# Session Start Playbook

Antes de trabajar en cualquier bloque:

1. Leer `CONTEXT.md`
2. Leer `docs/adr/DECISIONS.md`
3. Confirmar fase actual del roadmap
4. Resumir:
   - qué está construido
   - qué toca ahora
   - qué no se debe tocar aún
5. No introducir tecnologías fuera del stack confirmado
6. Trabajar en bloques pequeños y verificables
