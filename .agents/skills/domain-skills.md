# Domain Skills

## Validation phase principles
- el objetivo actual es validar interés
- la web es informativa primero
- el admin panel es ligero y funcional
- el sistema de torneos completo no entra aún

## Gedeon Web
Debe comunicar:
- regreso de marca
- nueva etapa
- visión futura
- llamado a la comunidad
- interés en participar

## Lead capture
Cada registro debe ayudar a responder:
- quién está interesado
- qué quiere jugar
- cuándo podría participar
- si quiere entrar al Discord

## Admin
Debe mostrar:
- total de registros
- formatos favoritos
- actividad reciente
- feedback resumido

## Discord
El bot debe apoyar:
- bienvenida
- captación
- encuestas simples
- check-ins futuros
