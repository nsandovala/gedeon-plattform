# Key Decisions

## D-001
PostgreSQL será la base de datos principal del sistema.

## D-002
FastAPI será la API central y futura fuente de verdad.

## D-003
Next.js será el frontend principal del sitio y paneles iniciales.

## D-004
Discord bot se desarrollará con Node.js + discord.js.

## D-005
Firebase / Firestore no se usará por ahora como base principal.

## D-006
Los agentes usarán:
- YAML para manifests/configuración
- Markdown para memoria compartida y playbooks

## D-007
La prioridad actual es validación de comunidad y marca, no automatización total de torneos.
