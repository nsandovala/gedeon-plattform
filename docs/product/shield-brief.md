# Shield Brief — Gedeon Esport

## Objetivo

Diseñar el escudo oficial de Gedeon Esport como símbolo central de identidad visual, adaptable a web, Discord, Instagram, favicon, ícono y futuras piezas promocionales.

El escudo debe sentirse:
- táctico
- geométrico
- sobrio
- moderno
- memorable
- premium
- vivo, no decorativo

## Qué debe representar

El escudo no es solo protección. Debe representar:

- guardia
- orden
- núcleo
- disciplina
- comunidad
- sistema activo
- regreso
- preparación para la batalla

Debe sentirse como una **insignia de una unidad táctica moderna**, no como una ilustración medieval.

## Dirección estética

### Concepto base
Un escudo minimalista y geométrico, construido con líneas limpias, proporciones elegantes y una presencia simbólica fuerte.

### Mezcla visual
- espartano moderno
- insignia táctica
- geometría sistémica
- tecnología contenida
- energía estructurada

## Rasgos formales deseados

- forma de escudo clara y simple
- contorno firme
- interior con uno o dos cortes geométricos significativos
- puede incluir un eje central o marcador vertical
- debe poder funcionar tanto en outline como en sólido
- debe ser reconocible incluso a tamaño pequeño

## Elementos que puede sugerir sin ser literal

- escudo espartano
- lanza o eje
- guardia
- centinela
- portal
- núcleo
- sistema
- órbita

## Elementos a evitar

- cascos espartanos literales
- alas
- espadas exageradas
- detalles barrocos
- formas medievales recargadas
- estética heráldica antigua
- adornos innecesarios

## Relación con el hero

El escudo no debe vivir aislado. Debe integrarse al hero como:
- emblema central
- marca de agua
- símbolo orbital
- núcleo visual del sistema

## Sistema orbital sugerido

El escudo puede estar acompañado por una estructura orbital abstracta inspirada en:
- órbitas
- agujero negro
- resonancia
- halo gravitacional
- campo energético

Esto debe sentirse elegante y matemático, no astronómico literal.

### Idea de comportamiento
- órbita circular o elíptica tenue
- líneas o puntos que rotan lentamente
- pequeños flickers o destellos rojos
- sensación de atracción al núcleo
- el escudo permanece estable mientras el campo alrededor respira

## Uso de color

### Primario
- rojo profundo mate
- humo blanco
- grafito oscuro

### Reglas
- el rojo debe resaltar el contorno o núcleo
- evitar brillos exagerados
- glow muy sutil y controlado
- preferir contraste limpio sobre saturación

## Versiones necesarias

### 1. Logo base
- escudo solo
- outline o sólido

### 2. Escudo con sistema orbital
- para hero, posters, animaciones o motion branding

### 3. App/icon version
- muy simplificada
- legible en pequeño

## Criterios de aprobación

El escudo final debe responder sí a estas preguntas:

- ¿Es reconocible a simple vista?
- ¿Se siente táctico?
- ¿Se siente moderno?
- ¿Se siente sobrio?
- ¿Tiene carácter propio?
- ¿Dialoga con la estética de AMON y tu lenguaje geométrico?
- ¿Podría funcionar como emblema de comunidad real?

## Frase guía

**No debe parecer un escudo dibujado.  
Debe parecer una señal de activación.**
