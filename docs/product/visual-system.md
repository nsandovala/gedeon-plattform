# Visual System — Gedeon Esport

## Propósito

Definir el sistema visual vivo de Gedeon Esport para que la marca no se sienta estática, genérica ni desconectada de su ADN geométrico y geek minimalista.

Este documento complementa el brand guide y aterriza:
- motion
- energía
- geometría
- ritmo visual
- firma gráfica
- sistema del hero

## Identidad visual extendida

Gedeon Esport no es solo una marca táctica.
También es un sistema visual activo.

Debe sentirse como:
- geometría viva
- energía contenida
- resonancia
- órbita
- estructura inteligente
- disciplina con pulso

## Sistema conceptual

### Capa 1 — Base táctica
- negro carbón
- grafito mate
- tipografía contundente
- bloques sobrios
- rojo con intención

### Capa 2 — Geometría
- mallas diagonales o radiales
- nodos
- líneas de campo
- patrones estructurados
- composiciones con centro de gravedad

### Capa 3 — Energía
- halos rojos suaves
- pulsos
- microdestellos
- respiración de opacidad
- glow tenue

### Capa 4 — Firma
- escudo
- órbita
- anillos
- eje central
- núcleo geométrico

## Motion language

### Principios
- lento
- sutil
- controlado
- elegante
- funcional
- casi imperceptible al principio, pero vivo

### Efectos válidos
- pulso lento de nodos
- órbita suave
- glow que respira
- líneas que cambian apenas de intensidad
- parallax mínimo
- aparición suave al scroll
- flicker eléctrico muy controlado

### Efectos prohibidos
- rayos exagerados
- chispas aleatorias
- humo
- glitches agresivos
- partículas sin sentido
- loops rápidos
- fondos ruidosos

## Hero signature system

El hero debe ser la pieza más viva y memorable de la web.

### Componentes del hero
1. fondo oscuro base
2. grid o malla geométrica
3. glow o halo rojo contenido
4. escudo principal
5. sistema orbital alrededor del escudo
6. nodos o marcas sutiles
7. headline monumental
8. microtexto operativo
9. CTA primario y secundario

## Fondo del hero

Debe sentirse como un campo activo.

### Puede incluir
- diagonales
- nodos
- líneas de campo
- una malla fina
- leves marcas rojas
- zonas de sombra y luz

### Debe evitar
- parecer un wallpaper
- competir con el texto
- quedarse muerto y decorativo

## Orbital system

La órbita del escudo es una de las posibles firmas más fuertes de Gedeon.

### Debe sentirse como
- campo gravitacional
- preparación
- alineación
- núcleo activo
- ritual táctico

### Visualmente
- anillo incompleto
- doble trazo
- puntos nodales
- halo externo
- ligera distorsión o respiración
- rotación muy lenta

## Tesla-like micro energy

Inspiración:
- electricidad elegante
- descarga controlada
- sistema en tensión
- resonancia mínima

### Uso
- pequeños destellos sobre nodos
- ligeras vibraciones de línea
- acentos rojos efímeros
- no más de 1 o 2 capas activas a la vez

## Scroll behavior

La web no debe sentirse muerta al bajar.

### Scroll language
- aparición por bloques
- opacidad escalonada
- desplazamiento suave de elementos gráficos
- cambio leve de intensidad del fondo
- microparallax en números, símbolos o anillos
- variación de glow en puntos clave

## Graphic rhythm

Las secciones deben alternar:
- bloque manifiesto
- bloque sistema
- bloque principios
- bloque despliegue
- bloque cierre ceremonial

No todas deben tener el mismo tono o densidad visual.

## UI energy rules

- el rojo no debe saturar la pantalla
- cada glow debe tener propósito
- toda animación debe sentirse intencional
- menos elementos, mejor resueltos
- el silencio visual también es parte del sistema

## Final principle

Gedeon no debe parecer una landing estática de esports.

Debe parecer:
- una marca en reactivación
- un sistema táctico con núcleo vivo
- una comunidad que vuelve con energía estructurada
- una mezcla entre disciplina espartana, geometría inteligente y pulso geek minimalista
